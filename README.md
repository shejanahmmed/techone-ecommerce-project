**TechOne** is a premium, high-performance tech e-commerce storefront designed for tech enthusiasts. It features a sleek, dark-themed UI with real-time shopping functionality, interactive product displays, and a streamlined user journey from discovery to checkout.

## 🚀 Live Demo
Experience the storefront live:

---

## ✨ Key Features

- **🔥 Dynamic Flash Sales**: Dedicated landing page with live countdown timers and stock urgency indicators.
- **🛒 Real-Time Cart System**: Fully functional Add-to-Cart, Wishlist, and Comparison features powered by `localStorage`.
- **🔍 Smart Search**: Global autocomplete search with instant product filtering and highlighted results.
- **📱 Responsive Design**: Fully optimized for desktop, tablet, and mobile screens.
- **💎 Premium UI/UX**:
  - Interactive product Quick-View modals.
  - Smooth glassmorphism effects and micro-animations.
  - Re-designed horizontal navbar with polished icon containers and glowing accents.
  - Square-shaped category grids for modern visual consistency.
- **🛠 Advanced Filtering**: Sort and filter products by category, price, and specs on dedicated shop pages.

---

## 🛠 Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3 (Custom Design System with Variables)
- **Icons**: FontAwesome 6.x
- **Typography**: Inter (Google Fonts)

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shejanahmmed/techone-ecommerce-project.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd techone-ecommerce-project
   ```

3. **Launch the site:**
   - Simply open `index.html` in any modern web browser.
   - Or use a local development server like VS Code's "Live Server".
